# VChat

## 项目简介

VChat 旨在在 Web 上提供一个能让你自由地与一个 VRoid 模型进行语音对话的方式。

# 功能特点

* [X] 与 VRoid 的进行语音聊天
* [X] 内置基础的 VRoid 角色
* [X] 多种有趣的聊天场景
* [ ] 自定义背景
* [ ] 自定义 VRoid 角色
* [ ] 自定义底层的聊天模型

# 开始

```bash
npm install
npm run dev
```

打开浏览器，访问 [http://localhost:3000](http://localhost:3000)。
